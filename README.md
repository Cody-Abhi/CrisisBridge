# Crisis Bridge

**Crisis Bridge** is a real-time, multi-role emergency response and crisis coordination platform purpose-built for the hospitality industry. It creates an instantaneous communication bridge between guests, hotel staff, and administrators during emergencies.

## Core Features

- **Real-Time SOS**: Trigger alerts in under 2 seconds.
- **Color-Coded Alerts**: Instant categorization of fire, medical, security, and other emergencies.
- **Unified Communication**: Integrated chat and WebRTC call systems.
- **AI Safety Assistant**: Gemini-powered guidance for guests in distress.
- **Admin Dashboard**: Bird's-eye view of the entire hotel's safety status.

## Technologies

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Realtime Database, Hosting)
- **AI**: Google Gemini 1.5 Flash
